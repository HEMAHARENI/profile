ROS2 SLAM using RTAB-Map

An advanced mobile robotics project implementing Simultaneous Localization and Mapping (SLAM) using RTAB-Map within the ROS2 framework. This sophisticated system integrates multiple sensors including depth cameras, odometry, and IMU to create real-time 3D maps while simultaneously tracking the robot's position in unknown environments.

The project demonstrates cutting-edge robotics navigation capabilities with sensor fusion techniques, real-time data processing, and advanced mapping algorithms. The system utilizes an Arducam Time-of-Flight (TOF) camera for precise depth data acquisition and UDP communication protocols for efficient data transmission between the robot and processing unit.

This implementation showcases practical applications of SLAM technology in autonomous navigation, with potential applications in service robotics, and exploration systems. The project is currently in final stages with localization improvements showing promising results.

Key Features:

  • Real-time 3D mapping using RTAB-Map algorithm
  • Multi-sensor fusion (Depth Camera, Odometry, IMU)
  • Arducam TOF camera integration for accurate depth perception
  • UDP communication protocol for efficient data transmission
  • Real-time depth data conversion to uint16 format for RTAB compatibility
  • ROS2 topic-based architecture for modular sensor integration
  • Simultaneous localization and mapping in unknown environments
  • RGB-D data processing for enhanced visual mapping
  • Loop closure detection for map consistency
  • Real-time visualization and monitoring capabilities

Technical Implementation:

  • Sensor Integration: Combined IMU, odometry, depth, and RGB data streams
  • Data Processing: Real-time conversion of depth data to RTAB-Map requirements
  • Communication: UDP port communication for camera data transmission
  • Mapping Algorithm: RTAB-Map for robust 3D environment reconstruction
  • Framework: ROS2 ecosystem for distributed robotics applications
  • Visualization: Real-time 3D map visualization and robot tracking

Project Status & Results:

  • Successfully integrated multiple sensor modalities for comprehensive mapping
  • Achieved real-time depth data processing and transmission
  • Implemented robust sensor fusion algorithms
  • Generated initial 3D mapping results with RTAB-Map integration
  • Currently finalizing localization improvements with enhanced accuracy
  • Final testing phase showing significantly improved mapping precision
  • Documentation and video demonstrations available upon project completion
